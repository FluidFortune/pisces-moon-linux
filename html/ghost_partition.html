SETUP WIZARD — Create New Ghost Partition
// SETUP OVERVIEW: Creates an encrypted container file that mounts as a filesystem. Uses LUKS (Linux Unified Key Setup) for maximum security. Container appears as a random binary file to casual inspection.
1
ALLOCATE CONTAINER SIZE
Choose a size for the hidden container. Appears as a file on your system.
dd if=/dev/urandom of=/home/pisces/.ghost.bin bs=1M count=1024 status=progress
2
FORMAT WITH LUKS
Initialize the LUKS encrypted container. You will be prompted for a passphrase.
sudo cryptsetup luksFormat /home/pisces/.ghost.bin
3
OPEN AND FORMAT
Open the container, create a filesystem, then close it.
sudo cryptsetup open /home/pisces/.ghost.bin ghost_vol sudo mkfs.ext4 /dev/mapper/ghost_vol sudo cryptsetup close ghost_vol
4
ADD AUTOSTART (OPTIONAL)
Add Pisces Moon edge bridge integration so the partition auto-prompts on boot with PIN screen.
echo "GHOST_CONTAINER=/home/pisces/.ghost.bin" >> ~/.pisces-moon/config
