PISCES MOON
FIELD INTELLIGENCE PLATFORM
OS v0.3 — DEBIAN 13 BUILD
A custom intelligence and field operations platform built on Debian 13 + XFCE 4.18, running on a Fujitsu Stylistic Q508 ruggedized tablet. Three-tier ecosystem connecting edge sensors, local compute, and cloud AI for wireless analysis, navigation, communications, and situational awareness.

THREE-TIER ECOSYSTEM
// TIER 1 — EDGE SENSOR (T-DECK)
LilyGO T-Deck Plus · ESP32-S3 · LoRa SX1262 · GPS NEO-6M · WiFi wardrive · BLE scanner · Keyboard + display. Runs wardrive.cpp, GPS tracking, BLE radar, LoRa mesh. Connects via USB serial → edge_bridge.py → WebSocket localhost:5006.
// TIER 2 — LOCAL COMPUTE (Q508 / Pi 400)
This device. 37 Pygame apps + full HTML app suite. Wardrive analysis, mapping, AI inference, file management, signal processing. Runs everything offline-capable. Edge bridge relays T-Deck data to HTML apps via WebSocket.
// TIER 3 — CLOUD AI (GEMINI)
Google Gemini 2.5 Flash via API. Optional intelligence layer. Baseball intel, trail finder, medical reference, survival consultant, general AI terminal. API key stored at ~/.pisces-moon/gemini.key. Works without connectivity for local-only operations.
