THREE-TIER ECOSYSTEM
// TIER 1 — EDGE SENSOR (T-DECK)
LilyGO T-Deck Plus · ESP32-S3 · LoRa SX1262 · GPS NEO-6M · WiFi wardrive · BLE scanner · Keyboard + display. Runs wardrive.cpp, GPS tracking, BLE radar, LoRa mesh. Connects via USB serial → pm_bridge.py → WebSocket ws://127.0.0.1:8080.
// TIER 2 — LOCAL COMPUTE (Q508 / Pi 400)
This device. Full HTML app suite (77 apps). Wardrive analysis, mapping, AI inference, field reference, signal intelligence. Runs everything offline-capable. pm_bridge.py relays T-Deck data to HTML apps via WebSocket on ws://127.0.0.1:8080.
// TIER 3 — CLOUD AI (GEMINI)
Google Gemini 2.5 Flash via API. Optional intelligence layer. Baseball intel, trail finder, medical reference, survival consultant, general AI terminal. API key stored at ~/.pisces-moon/gemini.key. Works without connectivity for local-only operations.
